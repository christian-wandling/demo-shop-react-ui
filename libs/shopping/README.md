# @demo-shop-react-ui/shopping

This library handles all aspects of the shopping feature including product selection, cart management, and the checkout process.

## Features

- **Shopping Cart Management**

  - Add/remove items
  - Update quantities

- **Checkout Process**

  - Address management
  - Order review
  - Order confirmation
